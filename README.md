# 🏛️🤖 Agência de IA — Pacote Completo

> **Tudo numa sequência só.** A Base (Opensquad + NEXUS + squads), a fábrica de conteúdo
> (fal.ai + Higgsfield), o rastreamento, o CRM Funnels, o Meta Ads, o Google Ads, o Windsor/GMB e os
> funis + cockpits. Você instala do zero, módulo por módulo, guiado pela IA — e termina com a
> sua agência de IA inteira no ar.

Template — gere a sua cópia e instale com o Claude Code, sem programar.

## 🚀 Como começar (3 passos)

1. **`Use this template`** → `Create a new repository`.
2. **Baixe:** `git clone` + `cd`. (Windows? [Tutorial Windows](docs/windows.md).)
3. **Abra o Claude Code e diga `instalar`** (`claude` → **`instalar`**).
   O Claude conduz a **sequência completa**, com um checkpoint entre cada módulo. Aula em
   [`aula/roteiro.md`](aula/roteiro.md).

## 🗺️ A sequência

1. **Base** (Opensquad) — o alicerce (atualiza o framework + skills, sem perder o que você criou).
2. **Meta Ads** — acesso de API à Meta (evolui o agente de Tráfego pra publicar).
3. **Google Ads** — MCC + API (comece cedo: Basic Access ~3 dias).
4. **Funil & Cockpits** — quiz, webinar, consultoria, LP + dashboards.
5. **CRM Funnels** — seu CRM conectado (recebe os leads do funil).
6. **Tracking** — rastreamento server-side no seu Cloudflare (fecha a atribuição).
7. **SEO Content** — fábrica de imagens/vídeos com IA + dashboard de aprovação.
8. **Windsor/GMB** — dados do Google Meu Negócio.

## ✅ Pré-requisitos (vá abrindo as contas)

GitHub · Node · Python · Cloudflare · CRM Funnels · fal.ai · Higgsfield · Meta Business Manager ·
Google Ads MCC + Google Cloud · Windsor.ai. O instalador avisa o que falta em cada etapa.

## 📦 O que tem aqui

Cada módulo vive em [`modulos/`](modulos/) self-contained (com a própria skill, aula e docs).
Quer instalar só um? Abra a pasta dele e diga "instalar". Quer tudo em sequência? Diga "instalar"
na raiz e a IA conduz.
