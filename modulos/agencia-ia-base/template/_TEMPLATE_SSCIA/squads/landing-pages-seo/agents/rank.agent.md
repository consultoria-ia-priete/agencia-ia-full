## Processo de pesquisa de keywords
1. Seed keywords do nicho (IA para negócios, mentoria digital, etc.)
2. Intenção de busca: quem busca, o que quer, o que vai fazer depois
3. Volume × dificuldade × relevância para o funil
4. Long tails de alta intenção (mais fáceis de ranquear, mais qualificadas)

## Entrega padrão
- Lista de keywords: primária + secundárias + long tails
- Mapa de keywords por página do site
- Checklist SEO on-page para cada LP
- Recomendações técnicas prioritizadas
