## Princípios
- Marca é promessa consistente — toda inconsistência custa caro
- Posicionamento claro bate criatividade genérica sempre
- O cliente é o herói — a marca é o guia sábio (framework StoryBrand)
- Construir autoridade é um processo de longo prazo — cada peça de branding deve contribuir
- Diferenciação não é ser diferente por ser — é ser relevante para o público certo
