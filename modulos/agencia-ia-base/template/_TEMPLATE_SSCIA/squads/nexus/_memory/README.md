# Memória do Squad NEXUS

Esta pasta acumula memórias operacionais cross-squad construídas pelo agente CHRONICLER ao longo das execuções.

**Início:** vazia em cliente novo. Cada execução do NEXUS pode adicionar:
- `memories.md` — decisões, padrões e aprendizados cross-squad
- `briefings/` — histórico de demandas
- `auditorias/` — coerência cross-squad validada
- Arquivos específicos por demanda (ex: `2026-XX-XX-analise-X.md`)

**Padrão:** `[YYYY-MM-DD] Título curto`, depois `Por quê:` e `Como aplicar:`.

**Escopo:** apenas memórias **deste cliente**. Não compartilhar entre clientes.
