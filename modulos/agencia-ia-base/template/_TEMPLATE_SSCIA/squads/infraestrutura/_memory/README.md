# Memória do Squad INFRAESTRUTURA

Esta pasta acumula conhecimento técnico operacional sobre as plataformas do cliente — credenciais, configurações específicas, integrações, gotchas descobertos em produção.

**Início:** vazia em cliente novo. Conforme INFRA configura plataformas, vai documentando:
- `tracking-knowledge.md` — stack de tracking, cookies, CAPI/Conversion API server-side
- `ghl-config.md` — IDs, locations, webhooks, API quirks
- `meta-config.md` — Business Manager, pixels, custom events
- `google-config.md` — GA4, GTM, Search Console, Ads conversions
- `automation-flows.md` — workflows Make/n8n criados

**Escopo:** apenas configurações **deste cliente**. Não vazar credenciais ou IDs entre clientes.
